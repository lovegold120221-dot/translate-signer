How the Eburon Pipeline Works

Translating to sign language requires more than text generation. We convert language into spatial mathematics, mapped directly to a digital human in milliseconds.

1

ASR & NLP Mapping

Dutch speech is transcribed via Whisper and converted into VGT structural gloss using our fine-tuned LLMs.

2

Motion Synthesis

Gloss is converted into animation curves (translation/rotation data) for hands, body, and non-manual facial markers.

3

Cloud Rendering & Streaming

NVIDIA Cloud GPUs render the full-body avatar in UE5 and stream the 1080p/60fps video feed directly to your client via WebRTC.

// Eburon WebRTC SDK Initialization
import { EburonStream } from '@eburon/vgt-sdk';

const avatarStream = new EburonStream({
    apiKey: 'eb_live_xxxxxxxx',
    region: 'eu-west',
    avatarProfile: 'sarah-professional',
    quality: '1080p60',
    bidirectional: true // Enables camera vision translation
});

// Connect to the DOM
await avatarStream.mount(document.getElementById('video-container'));

// Send Text to Sign
await avatarStream.sendInput({
    type: 'text',
    language: 'nl-BE',
    content: 'Welkom bij de klantenservice.'
});

// Listen for Customer Sign to Text
avatarStream.on('translation_received', (data) => {
    console.log("Customer Signed:", data.dutch_translation);
});
